# วิธีการติดตั้งและใช้งาน
วิธีการติดตั้ง <br>
-clone จาก github ไปบนเครื่อง<br>
-ให้ติดตั้ง python 3.6 ขึ้นไป<br>
-ติดตั้ง Tesseract OCR Engine และต้องตรวงสอบแก้โค้ด pytesseract.pytesseract.tesseract_cmd = r"C:\Program Files\Tesseract-OCR\tesseract.exe"
โดยการตั้ง Path ตามที่ตัวเองติดตั้ง tesseract.exe ไว้<br>
-Command Prompt (CMD) หรือ Terminal ในโฟลเดอร์ที่เก็บไฟล์โค้ด จากนั้นให้ผู้ใช้รันคำสั่ง pip install -r requirements.txt
<br>
วิธีการใช้งาน <br>
เมื่อกดรันหรือ กดเปิดไฟล์ exe แล้วจะไม่มีอะไรเด้งขึ้นบอกเลย <br>
ให้กด Ctrl + t เพื่อเป็นการเปิดหน้าต่างเลือกพื้นที่ที่ต้องการจะแปล <br>
ทำการคลิกและลากเมาส์ในพื้นที่ที่ต้องการ เหมือนกับการใช้ snipping tool <br>
เมื่อแอปแปลเสร็จจะเด้งหน้าต่างหลักขึ้นมา ที่จะแสดงข้อความที่ถูกแปล <br>
และเราสามารถพิมพ์ข้อความในภาษาได้และแปลเป็นภาษาอื่นได้ในช่องถัดมา <br>
สามารถซ่อนหน้าต่างนั้นได้โดยการกด esc จะเป็นการปิดหน้าต่างนั้นถ้าอยากให้เปิดต้องCtrl + tและแปลใหม่ <br>
วิธีการปิดแอปสามารถปิดได้2ช่วงคือ ช่วงที่เราอยู่หน้าแบบ snipping tool ให้กด esc หรือกดปุ่มออกโปรแกรมในหน้าต่างหลัก
